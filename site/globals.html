formatDate

  • formatDate(date?: Date | string, format?: string): string
    • Defined in format/index.tsx:35

    格式化日期对象

    example

    formatDate(new Date("2016/01/01 00:00:00")) // "2016/01/01 00:00:00"

    example

    formatDate(new Date("2016/01/01 00:00:00"), "yyyyMMdd") // "20160101"

    see

    https://docs.oracle.com/javase/7/docs/api/java/text/SimpleDateFormat.html

    Parameters

    • Default value date: Date | string = new Date()

      日期对象

    • Default value format: string = "yyyy/MM/dd HH:mm:ss"

      格式字符串，其中以下字符（区分大小写）会被替换：

      字符 意义 示例
      y 年 yyyy: 1999, yy: 99
      M 月 MM: 09, M: 9
      d 日 dd: 09, d: 9
      H 时（24小时制） HH: 13, H: 13
      m 分 mm: 06, m: 6
      s 秒 ss: 06, s: 6
      e 周（中文） 周e: 周一

    Returns string

    返回格式化后的字符串
